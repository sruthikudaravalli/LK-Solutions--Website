#LK-Solutions
Worked as a part of team in developing web application which provides solutions to Knapsack problem and Linear packing problem. Developed the front-end of the website by creating the webpages using HTML5, CSS3, jQuery and JavaScript.	
Environment: HTML5, PHP, CSS, Java Script, MySQL, Windows 8.

Link to view website:

http://moe.csci.unt.edu/~15SPUSMXHCIAI1/website/

While providing the inputs for both the problems please be sure that any of the item's weight is not more than container weight:

Example:  Linear packing problem Input
Weight limit per container: 5
Items' weight: 1 3 2 4 2

Knapsack Problem input
weight limit per container: 5
Items' weight: 1 3 4 2 1
Items' costs: 45 67 3 70 21
